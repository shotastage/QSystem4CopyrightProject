# QSystem for iPad

Rewritten q-system application for iPad user.


# Requirement

## Basic Requirement

- **macOS High Sierra < 10.3.5**
- **Xcode < 9.4**
- **iOS 11 on iPad**

# Installation

## 1. Install or select commandline-tools

## 2. Install homebrew

## 3. Install Cocoapods

## 4. Open `.xcworkspace` via Xcode

>> Currently, iPhone or old iPad is not supported!
